Casino Games: Rules, Probabilities, and Technical Details

1. Coinflip

Using a for loop, this will generate either a 0 or a 1 value, further incrementing:
0: heads++;
1: tails++;
Odds are programmed exactly 50/50, giving a fair distribution of values.

3. Roulette Payouts

  • Betting on an exact number pays out 35:1 (straight-up)
  • Betting on either red or black pays out 2:1
  • Betting on either odd or even pays out 2:1
  • Betting on 1-18 or 19-36 pays out 2:1
  • Betting on 1st/2nd/3rd twelve pays out 3:1

4. Dice Functionality and Payouts

Same as coinflip, generates 1-6 using a for loop, further incrementing:
1: diceOnes++;
2: diceTwos++;
3: diceThrees++;
4: diceFours++;
5: diceFives++;
6: diceSixes++;

5. Slot Machine Functionality and Payouts

Generates 3 values 1 through 9, each representing one slot in the machine:
- If one 7 value appears, the payout is 1.5x of the initial bet
- If two 7 values appear, the payout is 8x of the initial bet
- If all 3 slots have the value 7, this occurs as a jackpot and pays out 50x
- If 2 same values appear, the payout is 3x
- If all of the values are the same, the payout is 50x
